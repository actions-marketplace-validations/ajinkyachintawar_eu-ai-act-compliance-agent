GitHub Action — CI/CD Gate

Block PRs that introduce EU AI Act violations into your AI codebase. Only needs a Groq API key — no deployed backend required.

  1. 1

    Add your Groq API key as a repository secret

    GitHub repo → Settings → Secrets → Actions → New repository secret

    Name: GROQ_API_KEY
    Value: gsk_your_key_here     Get a free Groq API key →
  2. 2

    Create the workflow file

    Add this to .github/workflows/eu-ai-act-scan.yml 

    name: EU AI Act Compliance Scan

on:
  pull_request:
    paths: ["**/*.py"]

jobs:
  compliance-scan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: EU AI Act Compliance Scan
        uses: ajinkyachintawar/eu-ai-act-compliance-agent@main
        with:
          groq-api-key: ${{ secrets.GROQ_API_KEY }}
          files: "src/**/*.py"
          system-context: "Describe your AI system here"
          fail-on: CRITICAL   # CRITICAL | HIGH | MEDIUM | LOW | NONE

      - uses: actions/upload-artifact@v4
        if: always()
        with:
          name: eu-ai-act-scan-report
          path: eu-ai-act-scan-report.json
  3. 3

    Add the README badge

    Shows compliance scan status on your repo.

    [![EU AI Act Scan](https://github.com/YOUR_USER/YOUR_REPO/actions/workflows/eu-ai-act-scan.yml/badge.svg)](https://github.com/YOUR_USER/YOUR_REPO/actions/workflows/eu-ai-act-scan.yml)
